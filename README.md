# back-man-template-sys
后台管理系统模板

# 环境设置
1. 初始化npm代理
```bash
npm config set registry https://registry.npm.taobao.org
```

2. 安装 cnpm
```bash
npm install -g cnpm
```
[pnpm学习升级]


3. 进入根目录下，初始化依赖
```
cnpm i
cnpm run dev
```
4. 项目基本介绍
 ## ui框架：element
 ## eslint规范：airbnb
 ## css预解析器：less
 ## 路由模式：history
